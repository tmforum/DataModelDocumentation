## Data model

A JSON Schema corresponding to this data model can be found
[here](https://github.com/tmforum-rand/schemas/blob/candidates/Analytics/ThingAnalytics.schema.json).

The Data model is defined as shown below:
- `activationDate` : Date the thing was first used on the network
  - Optional
- `deactivationDate` : Date the thing was deactivated from the network
  - Optional
- `dataCreationTimestamp` : Time stamp for data creation (e.g. system dump creation, event generationâ€¦)
  - Optional
- `runTimestamp` : Time stamp for data upload run
  - Optional
- `validFromTimestamp` : Time stamp for business validity of entity record
  - Optional
